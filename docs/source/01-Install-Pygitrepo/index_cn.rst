1. 安装 ``pygitrepo`` 命令行工具
------------------------------------------------------------------------------

``pygitrepo`` 本身是一个 python 库, 只不过是一个自带命令行接口的库, 就和 ``pip``, ``pipenv``, ``virtualenv`` 类似. 所以你可以用 ``pip`` 命令安装.

.. code-block:: bash

    # install and upgrade to latest
    $ pip install pygitrepo --upgrade

    # test with ``pgr`` command
    $ pgr
